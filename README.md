# healthcrypt
Secure Online Patient Appointment Booking and Encrypted Medical History Management System

HealthCrypt is a secure web-based healthcare management system that enables patients to book appointments and maintain their medical history safely. The platform ensures data privacy and security through encryption and secure authentication mechanisms.

It provides a seamless interface for patients, healthcare staff (HSA), and administrators to manage appointments, patient records, and medical data efficiently.

🚀 Features
👤 Patient
Register and login securely,
Book appointments with doctors,
View and manage appointments,
Maintain personal medical history
🏥 Healthcare Staff (HSA)
Manage patient records,
Approve / cancel / reschedule appointments,
Add and manage doctors,
View reports and statistics
🔐 Security Features
Password hashing using bcrypt,
CSRF protection enabled,
Session-based authentication,
Secure data handling
